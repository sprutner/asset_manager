Asset Manager

A simple asset manager for spacecraft equipment.

https://github.com/sprutner/asset_manager

Adding a new asset to the database

Submit a POST request to http://:5000/assets with JSON data in the following format:

{ "name":"antarctica", "asset_type": "antenna", "asset_class": "yagi" }

Getting all assets

Submit a GET request to http://:5000/assets and you will receive an object containing all assets

Getting an asset by name

Submit a GET request to http://:5000/asset/name/ and you will receive that asset. e.g. http://localhost:5000/asset/name/antarctica

Getting an asset by asset_id

Submit a GET request to http://:5000/asset/ and you will receive that asset. e.g. http://localhost:5000/asset/1
